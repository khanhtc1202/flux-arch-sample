# Sample Flux architecture for React App

A sample React application implemented under (FLUX)[https://code-cartoons.com/a-cartoon-guide-to-flux-6157355ab207] architecture.

## Project structure

```bash
.src
├── actions
│   ├── SearchAction.ts
│   └── TypeAction.ts
├── components
│   ├── SearchResults.tsx
│   └── SearchBox.tsx
├── containers
│   ├── SearchBox.ts
│   └── SearchResults.ts
├── reducers
│   ├── Search.ts
│   ├── Type.ts
│   └── index.ts
├── index.tsx
├── state.ts
└── store.ts
```
